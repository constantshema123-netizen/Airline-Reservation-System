# Airline-Reservation-System
Built a console-based Airline Reservation System using Java, applying Object-Oriented Programming principles such as encapsulation and modular design. The system manages passengers, flights, and bookings with features like seat availability checking, fare calculation, and ticket generation.


# OBJECTIVES
- To practice Java programming
- To understand Object-Oriented Programming (OOP)
- To design a real-world system using classes
- To improve problem-solving and system structuring skills

- # FEATURES
- Passenger registration and validation
- Flight creation and seat availability checking
- Fare calculation with tax
- Booking system with seat and class selection
- Ticket generation
- Booking cancellation

# HOW TO RUN

1. Open the project in IntelliJ IDEA or VS Code
2. Compile all Java classes
3. Run the Main class
4. Enter input when prompted in the console
